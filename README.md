
# Repository

Um aplicativo mobile programado em React, exibe uma listagem com detalhes de personagens da aclamada série Rick And Morty




## Rodando localmente

Clone o projeto

```bash
  git clone https://github.com/jocxzs/Repository
```

Entre no diretório do projeto

```bash
  cd Repository
```

Instale as dependências

```bash
  npm install
```

Inicie o servidor

```bash
  npx run start
```
Leia o QR Code gerado no terminal pelo aplicativo EXPO GO,
Ou utilize o Web bundler para vizualização web (Podem haver diferenças na forma como o aplicativo será mostrado na web e no mobile, ou diferença entre navegadores)
